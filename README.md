# LetsEat 🍲

## General info

LetsEat is a restaurant reservation app created with Swift and Xcode.


This app has been created:
 
✅ UIKit

✅ MVC Architecture

✅ CoreData

✅ AutoLayout

✅ Static TableView

✅ Custom UIControl

✅ MapKit

✅ AVFoundation
